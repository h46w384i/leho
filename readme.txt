         程序版本
----------------------------------
佳弗O2O网站系统——乐活版
 
        环境需求
----------------------------------
1. 可用的 www 服务器，如 Apache、Zeus、IIS 等
2. php 4.3.0 及以上
3. Zend Optimizer 2.5.7 及以上
4. MySQL 3.23 及以上，仅针对 MySQL 版

           安装
----------------------------------
1.安装按照提示流程一步一步安装
2.安装过后如果出现错误，请删除 ./Runtime文件夹
3.如果打开首页 点击导航 出现404错误 请修改 config.php的值 URL_MODEL改为3;
